#  ShopZone

ShopZone is a modern ecommerce web application built using React and Vite.  
The project provides a smooth shopping experience with cart management, checkout flow, order tracking, authentication system, and responsive UI.

---

##  Features

-  User Login & Signup
-  Guest Login Support
-  Product Browsing
-  Search & Category Filters
-  Cart Management
-  Quantity Controls
-  Checkout System
-  Order Tracking
-  Previous Orders History
-  Customer Support Page
-  Fully Responsive Design
-  Modern UI with Hover Effects

---

##  Technologies Used

- React.js
- Vite
- React Router DOM
- Context API
- CSS3
- Local Storage
